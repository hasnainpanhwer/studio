# **App Name**: PageEdge

## Core Features:

- Image Capture/Upload: Capture images using the device camera or upload from the gallery.
- Automatic Edge Detection & Cropping: Automatically detect page boundaries in images and crop the unnecessary regions. Includes tool to use the LLM to estimate border thickness and darkness to improve results.
- Perspective Correction & Skew Alignment: Correct perspective distortions and align skewed pages for a clean, top-down view.
- Shadow Removal & Contrast Enhancement: Clean up shadows, reduce lighting distortions, and enhance contrast to improve readability.
- Margin Adjustment & Border Creation: Trim excessive margins and optionally add custom side borders to mimic a neatly framed page.
- OCR & Text Extraction: Run OCR (Optical Character Recognition) to extract editable text from the cleaned image, preserving the layout without altering content areas.
- Export Options: Allow users to preview cleaned images and export them as individual images or combined PDFs with recognized text.

## Style Guidelines:

- Primary color: Soft beige (#F5F5DC) to give the application a vintage paper feel.
- Background color: Off-white (#FAFAFA), very slightly desaturated from the primary color.
- Accent color: Light brown (#B8860B), to contrast the background.
- Headline font: 'Literata', a transitional serif that pairs a literary feel and contemporary sensibilities.
- Body font: 'Inter', a grotesque-style sans-serif with a modern feel.
- Simple, clean icons representing capture, edit, and export functions.
- Clean and intuitive layout optimized for mobile and tablet screens.
